## build adn run full-node (server) (node version >= 12)

1. install deps
   ```
   npm install
   ```
1. compile saito
   ```
   npm run nuke
   ```
   Might take a while. Wait till `Bundle Success!`
1. run local dev
   ```
   npm start
   ```
   http://localhost:12101/

## dev workflow for light-node (client)

1. run client
   ```
   npm run client
   ```
   http://localhost:8888/
